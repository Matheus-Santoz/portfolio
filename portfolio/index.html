Sobre mim

Me chamo Matheus Santos, tenho 19 anos e moro na zona leste da cidade de São Paulo, no bairro de Guaianases. Desde cedo, sou apaixonado por tecnologia e encontrei no desenvolvimento de sistemas a minha verdadeira vocação. Gosto de desafios e estou sempre em busca da melhor solução para cada problema.

Concluí o ensino médio técnico em Análise e Desenvolvimento de Sistemas na ETEC de Guaianases e, atualmente, estou cursando o tecnólogo na Fatec São Paulo, no 1º semestre. Sigo aprendendo e me aprimorando todos os dias para construir soluções criativas e eficientes.

Com o que eu trabalho?

Além de ter adquirido outras softskills em empresas anteriores, como trabalho em equipe e uma boa comunicação:

HTML

Foi o primeiro conceito que aprendi nessa trajetória, um dos que mais reforço, considero como um arroz e feijão e ando sempre estudando e praticando.

CSS

Reforço tanto quanto o HTML, busco aprender mais sobre suas propriedades para ter o layout responsivo da maneira mais limpa possível. Esse site se adapta á qualquer dimensão, pode testar!

JavaScript

Uso JavaScript para adicionar interatividade aos meus projetos, tornando-os mais dinâmicos e animados, tipo esse carrosel das habilidades que você está vendo agora :)

Python

Usei Python durante meu curso técnico, principalmente para automação de tarefas e desenvolvimento de scripts simples. Estou voltando a usar no tecnólogo.

Usei PHP para fazer o back-end do meu TCC, incluindo sistemas de login, cadastro, autenticação, estoque e sistema de pagamento.

Usei o Git durante muito tempo para facilitar o versionamento dos meus projetos, o que me ajudou a evitar problemas com versões, principalmente trabalhando em equipe.

MySQL

Também tenho experiência com MySQL, utilizei durante os 3 anos do meu curso técnico para armazenar e manejar dados de forma eficiente em projetos pessoais e acadêmicos. Foi o banco que usei também no TCC.

VBA

Além de ter sido componente do meu curso, busco sempre aprender mais, tanto para melhorar a usabilidade dos meus projetos quanto para criar layouts satisfatórios, porém ainda estou caminhando.
